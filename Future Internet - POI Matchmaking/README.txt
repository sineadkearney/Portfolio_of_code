Point of Interest Matchmaking Specific Enabler.

This enabler allows any number of players connected to the server to group together based on their spatial proximity to a point of interest.

The documentation for this project can be found at: http://mediafi.org/?portfolio=geospatial-poi-matchmaking

The github repository for this project can be found at: https://github.com/fi-content2-games-platform/FIcontent.Gaming.Enabler.PoIMatchmaking
